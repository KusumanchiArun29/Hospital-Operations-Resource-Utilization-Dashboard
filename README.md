# Hospital Operations & Resource Utilization Dashboard

## Project Overview
This Power BI project analyzes hospital operations to monitor patient flow,
bed utilization, and treatment costs across departments.

## Key Features
- Executive KPI overview
- Patient distribution by department and doctor
- Emergency vs OPD admission analysis
- Treatment cost and bed utilization insights
- Interactive filters and slicers

## Tools & Skills
- Power BI
- DAX
- Power Query
- Data Modeling
- Data Visualization
